# Campus Clubs and Societies Recruitment System<br>
This project automates student recruitment for college clubs.
<br>
Technologies Used:
- C++
- OOP
- Trie Data Structure
- Priority Queue (Max Heap)
- JSON File Handling
- HTML, CSS, JavaScript
<br>
Features:
- Fast club searching using Trie
- Merit-based ranking using Max Heap
- Tier-based membership system
